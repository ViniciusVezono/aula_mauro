1. text-align

A propriedade text-align alinha o texto.

Valores: start, end, left, right, center, justify, justify-all, or match-parent.

2. text-align-last

A propriedade text-align-last ajusta como a última linha do texto fica.

Mesmos valores de text-align.

3. Exercício

  1. Ajuste o alinhamento do texto no corpo da página css_aula2_exemplo13.html para justify.
  2. Ajuste o alinhamento da última linha dos parágrafos na página css_aula2_exemplo13.html para right.

4. vertical-align

Ajusta o alinhamento vertical de um elemento com os outros.

Opções: https://developer.mozilla.org/en-US/docs/Web/CSS/vertical-align 

span {
  vertical-align: super;
}

5. text-decoration

A propriedade text-decoration ajusta o tipo de linha (obrigatório), a cor, o estilo e espessura da linha.

Podemos ajustar separadamente com:

  • text-decoration-line
  • text-decoration-color
  • text-decoration-style
  • text-decoration-thickness

Ou de uma vez com:

  • text-decoration

5.1. line

Os valores para text-decoration-line estão aqui.

Mais usados: none, underline e line-through.

5.2. color

Podemos usar qualquer cor HTML como valor para text-decoration-color.

5.3. style

Os valores solid, double, dotted, dashed e wavy são aceitos por text-decoration-style e outros.

5.4. link

Para retirar o sublinhado de links use: 

a {
  text-decoration: none;
}

6. text-transform

Podemos exibir o texto diferente do conteúdo usando text-transform. Valores:

capitalize
Primeira Letra Maiúscula
uppercase
LETRAS MAÍUSCULAS
lowercase
letras minúsculas

Existem outros valores. 

.nome {
  text-transform: capitalize;
}

7. Espaço no texto

Podemos controlar a identação usando text-indent.

O espaço entre as letras é ajustado com letter-spacing.

O espaço entre as linhas é ajustado com line-height.

O espaço entre palavras é ajustado com word-spacing.

A propriedade white-space ajusta como o espaço em branco aparece, nowrap não quebra e pre quebra em <br> e nova linha.

8. sombra

A propriedade text-shadow adiciona sombra ao texto.

Sintaxe: 

text-shadow: posX posY raio cor;
posX
é a distância na horizontal da sombra ao texto original (obrigatório)
posY
é a distância na vertical da sombra ao texto original (obrigatório)
raio
é o tamanho do borrado da sombra.

9. fonts

Famílias de fontes genéricas:

  1. Serif - elegância.
  2. Sans-serif - moderno.
  3. Monospace - datilografado.
  4. Cursive - escrita à mão.
  5. Fantasy - decorativa.

Propriedade font-family ajusta a família. 

p {
    font-family: "Times New Roman", Times, serif;
}

Comece com a fonte que deseja, se o navegador não encontrar usará a próxima. Deixe no final a família genérica.

9.1. web safe

Algumas fontes estão em todos os dispositivos (web safe):

  • Arial (sans-serif)
  • Verdana (sans-serif)
  • Tahoma (sans-serif)
  • Trebuchet MS (sans-serif)
  • Times New Roman (serif)
  • Georgia (serif)
  • Garamond (serif)
  • Courier New (monospace)
  • Brush Script MT (cursive)

9.2. Font style

A propriedade font-style pode aplicar itálico em um texto: 

p {
    font-style: italic; /* outras opções: normal e oblique
}

9.3. font weight

A propriedade font-weight ajusta a espessura do texto. Valor bold é o texto espesso. 

p {
    font-weight: bold;
}

9.4. Font-variant

É possível exibir o texto de maneira diferente com font-variant: small-caps;

9.5. Font-size

Font-size ajusta o tamanho da fonte.

Podemos ajustar um valor em px, de acordo com o tamanho da letra M atual 5em ou de acordo com a largura da janela 10vw; 

p {
    font-size: 10vw;
}

1vw = 1% da largura da tela.

9.6. Font resumida

A propriedade font ajusta de uma vez:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

Os valores font-size e font-family são obrigatórios.

11. Links

Podemos ajustar os diferentes estados dos links:

a:link
ajustar o link não visitado.
a:visited
ajustar o link que já foi visitado.
a:hover
ajustar o link quando o mouse está sobre ele.
a:active
ajustar o link quando o mouse está apertado.

A ordem de ajuste é:

a:hover após a:link e a:visited. a:active após a:hover.

Podemos retirar o underline com text-decoration: none;

12. Display

Podemos controlar como o elemento aparece na página usando display.

O valor padrão é: block ou inline.

display: block;
começa em nova linha, ocupa toda a largura.
display: inline;
ocupa apenas o espaço do conteúdo.

Outros valores:

https://developer.mozilla.org/pt-BR/docs/Web/CSS/display
